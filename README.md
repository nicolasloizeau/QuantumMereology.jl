# QuantumMereology


[![Dev](https://img.shields.io/badge/docs-dev-blue.svg)](https://nicolasloizeau.github.io/QuantumMereology.jl/dev/)
[![Build Status](https://github.com/nicolasloizeau/QuantumMereology.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/nicolasloizeau/QuantumMereology.jl/actions/workflows/CI.yml?query=branch%3Amain)

Tools to solve the following problem:
Given a Hamiltonian $H$ and a set of Pauli strings $\{P_i\}$, find a unitary $U$ such that $U H U^\dagger$ is a linear combination of the $P_i$.
